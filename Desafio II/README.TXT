---------------------------------------------------------------------------------

Esse foi o segundo programa em Java feito sozinha, a proposta era fazer um 
programa para uma biblioteca de cursos onde qualquer um poderia acessar e adicionar
um curso novo, preenchendo todos os dados pedidos.

---------------------------------------------------------------------------------

TÍTULO DO PROJETO: Biblioteca de Cursos
OBJETIVO DO PROJETO: Cadastro de cursos pelo cliente

---------------------------------------------------------------------------------

Foi usado a classe Teclado, fornecida pela própria faculdade para substituir a 
classe Scanner.
