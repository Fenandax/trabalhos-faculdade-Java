---------------------------------------------------------------------------------

Esse foi um projeto para o fórum da faculdade, de tema livre e a minha proposta 
foi fazer um programa para o professor reservar salas para aulas presenciais 
durante a pandemia. A proposta é que cada sala teria uma quantidade limitada de 
lugares com distanciamento e na hora do cadastro ele poderia não ser aprovado por 
conta da quantidade de pessoas ou receber um aviso se tivesse a quantidade máxima.

---------------------------------------------------------------------------------

TÍTULO DO PROJETO: Reserva de salas
OBJETIVO DO PROJETO: Reserva de salas para ser usado pelo professor

---------------------------------------------------------------------------------

Foi usado a classe Teclado, fornecida pela própria faculdade para substituir a 
classe Scanner.
